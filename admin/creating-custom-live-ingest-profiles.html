Création de profils d'ingestion en direct personnalisés

Dans cette rubrique, vous apprendrez comment créer et gérer des profils d'ingest Live personnalisés.

La page Administration des profils d'ingest permet de gérer les profils d'ingest en direct personnalisés. Les profils d'ingestion en direct déterminent le nombre et la qualité des formats associés créés par Brightcove Live. Les profils Live ingest ne sont pas utilisés lors du téléchargement de vidéos vers un compte Video Cloud. Les profils d'ingest sont définis à l'aide de JSON. Avant de créer un nouveau profil d'ingestion, consultez le document Profils d'ingest standard pour voir s'il existe un profil existant qui répondra à vos besoins.

Notes :
  • Chaque profil peut créer un maximum de 25 formats associés.
  • Vous pouvez créer des rendus pour les vidéos verticales, il suffit d'inclure height et de width valeurs dans le spécifications de rendu qui correspondent à la ration d'aspect de la vidéo (1080 pixels est la valeur maximale autorisée pour le height).
  • Lorsque les profils en direct sont masqués dans Studio sous Administrateur > Ingérer des profils , ceux-ci ne seront pas sélectionnables dans le Créer un événement en direct UI mais ils peuvent toujours être utilisés avec l'API Live.
  • Niveaux en direct :
    • Pour qu'un profil d'ingestion Live soit considéré comme Tier 1, la somme des débits binaires de tous ses rendus ne peut pas être supérieure à 5 450.
    • Si la somme des débits binaires de tous ses rendus est supérieure à 5 450 mais inférieure à 10 000, le profil d'ingestion en direct sera considéré comme de niveau 2.
    • Si la somme des débits binaires de tous ses rendus est supérieure à 10 000, le profil sera considéré comme de niveau 3.
Chaque profil peut créer un maximum de 25 rendus.

Création d'un nouveau profil

La page Administration des profils d'ingest peut être utilisée pour créer des profils d'ingest en direct personnalisés. Notez que pour que le profil apparaisse dans l'interface utilisateur du module Live, il doit avoir dans le "model_version": 1 cadre des propriétés du profil.

Pour créer un profil d'ingestion en direct personnalisé :

  1. Cliquez sur le lien ADMIN dans l'en-tête Studio.
  2. Clique le Ingérer des profils relier.
  3. Cliquez sur Nouveau profil > en direct.
  4. Entrez un nom pour le profil, puis cliquez sur Créer.
  5. Utilisez l'Éditeur de profils pour modifier le profil selon vos besoins.
  6. Cliquez sur Save.

Duplication d'un profil

Il est possible de créer un nouveau profil en dupliquant un profil existant. Ensuite, vous pouvez modifier le nouveau profil avec vos modifications. Notez que pour que le profil apparaisse dans l'interface utilisateur du module Live, il doit avoir "model_version": 1 en tant que partie du profil.

Pour créer un profil d'ingestion en direct personnalisé en dupliquant un profil existant :

  1. Cliquez sur le lien ADMIN dans l'en-tête Studio.
  2. Clique le Ingérer des profils relier.
  3. Localisez un profil avec un Type de Standard Live à dupliquer. Cliquez sur le nom du profil pour ouvrir l'éditeur de profils.
  4. Cliquez sur Dupliquer. L'éditeur de profil s'ouvrira avec le nouveau profil.
  5. Faites défiler la page vers le haut du profil. Modifiez l' name élément pour donner au profil un nom significatif. Ce nom s'affichera dans Video Cloud Studio.
  6. Modifiez l' description élément pour donner au profil une description significative.
  7. Modifiez les informations de format associé si nécessaire.
  8. Cliquez sur Save.

Le profil nouvellement créé apparaîtra dans la liste des profils. Les profils personnalisés peuvent être supprimés en cliquant sur le bouton supprimer () bouton.
