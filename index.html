Brightcove Live

Le module Live est utilisé pour diffuser des événements en direct à la fois sur les ordinateurs de bureau et les appareils mobiles. Lorsque vous aurez saisi les informations sur l'événement dans le module Live, vous obtiendrez un ensemble de paramètres d'encodage à intégrer dans l'encodeur de votre site. La conversion se fait dans le nuage ; vous n'avez donc pas besoin d'un convertisseur matériel.

Pour les cas d'utilisation et les intégrations plus complexes, vous pouvez également utiliser l'API Live.
