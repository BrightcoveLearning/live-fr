Diffusez un événement en direct sur l'application Beacon

Dans cette rubrique, vous apprendrez comment diffuser un événement en direct sur l'application Beacon.

Présentation

Pour diffuser un événement en direct dans votre Application, vous devez suivre une série d'étapes pour connecter votre compte Video Cloud Live et Brightcove Beacon.

Une vue de haut niveau du processus est :

  1. La première étape consiste à créer un événement et une playlist dans Brightcove Beacon. Cet événement sera disponible dans l'application en tant que lieu de place pour votre vidéo.
  2. Accédez à votre compte Video Cloud et accédez à Live pour créer votre événement et obtenir le Job ID qui fonctionnera comme identifiant.
  3. De retour dans Brightcove Beacon, vous pourrez connecter votre streaming en direct en mettant l'ID de travail généré dans la plateforme Live dans la section Evénements de la plateforme.
  4. Attendez la connexion et la vidéo Live démarre en streaming dans l'application Beacon.
  5. Dans la dernière section, vous verrez les étapes à suivre pour créer un autre type d'événement dans Live by a Static Entry Point.

Le reste du présent document fournira des détails pour chacune de ces étapes.

Créez un événement en direct dans Beacon :

Dans la plateforme Brightcove Beacon, cliquez sur l'onglet Événements pour démarrer la configuration d'un événement :

Seuls les comptes activés avec Balise Brightcove en direct aura le Listes de lecture et Événements disponible dans la section onglets. Veuillez contacter le support Brightcove au cas où ils ne sont pas activés.
  1. Cliquez sur Ajouter un nouvel événement:
  2. Donnez un nom à l'événement Live et une date de début de l'événement:
  3. Dans le Données textuelles onglet, remplissez les informations de votre événement :
  4. Dans le Données non textuelles onglet, choisissez une affiche d'image pour votre événement :

    Vous pouvez sélectionner le type d'affiche que vous souhaitez pour votre événement entre Portrait, Paysage, Miniature et Large bannière.

  5. Dans le Droits et planification onglet, vous pourrez définir le Détails de disponibilité comme:
    • Type de droits : Choisissez le type de droits entre Publicité, Abonnement ou Gratuit.
    • Heure de début : L'heure de début de votre événement
    • Heure de fin : L'heure de fin de votre événement, ces heures sont créées pour générer des limites dans l'application.
    • Flux max : Nombre maximal de flux autorisés dans l'application. Le nombre par défaut est 0 (zéro).
    • Détails sur les appareils : Ici, vous pouvez ajouter le nombre différent d'appareils sur lesquels vous souhaitez que votre flux soit publié. (Exemple : iOS, Android, Roku, Samsung, etc.)
    • Détails de la géo-localisation : Liste des pays dans lesquels vous souhaitez activer votre application. Vous pouvez choisir « Monde » au cas où vous ne souhaitez pas restreindre un pays.
    • Emplacements refusés : Liste des régions où l'accès à votre contenu est refusé.
  6. Cliquez sur le bouton ci-dessous : Créer un nouvel événement

Ajouter l'événement Beacon à une playlist

Une fois que nous avons créé un événement, allez dans l'onglet Listes de lecture dans le menu supérieur. Nous devons ajouter l'événement à une playlist avant de pouvoir diffuser quelque chose.

  1. Cliquez sur le bouton Ajouter une nouvelle playlist et créez-en une.
    Vous devez créer une liste de lecture dans Brightcove Beacon qui contiendra vos événements en direct. N' utilisez pas de liste de lecture de Video Cloud, car l'utilisation d'une liste de lecture Video Cloud peut écraser et supprimer vos enregistrements Beacon Live.
  2. Dans Listes de lecture, dans les onglets Type de contenu, sélectionnez l'onglet Événements :
  3. Recherchez votre événement dans le module contenu et cliquez sur le bouton (+) pour l'ajouter à la liste de lecture de gauche. Vous pouvez positionner l'ordre des événements de la manière dont vous souhaitez qu'ils s'affichent dans l'application.
  4. Réglez les paramètres de votre playlist (heure, appareils, localisation, etc...) Comme vous l'avez fait avec votre événement.
  5. Clique sur le Enregistrer les modifications dans la liste de lecture bouton.
  6. Cliquez sur le bouton de l'outil en haut à droite de l'écran pour effacer le cache, de sorte que tout s'affiche correctement dans l'application.

À ce stade, votre événement sera créé en tant que conteneur dans l'application Brightcove Beacon, mais pour commencer à diffuser une vidéo en direct, nous devrons configurer notre transmission sur Video Cloud Live Platform.

Créer un événement Live dans Studio

Pour accéder au module Live, connectez-vous à Video Cloud Studio et cliquez sur Live dans l'en-tête de navigation.

  1. Sur la page Gérer les événements, cliquez sur Créer un événement. La page Créer un événement en direct ou Créer une chaîne en direct s'ouvre. Les propriétés lors de la création d'événements et de canaux sont les mêmes. La capture d'écran de ce document couvre la création d'un événement.
  2. Entrez les détails de l'événement :
    • Nom de l'événement : nom de l'événement ; ce nom apparaîtra également dans le module Media si la vidéo est enregistrée en tant que ressource vidéo (255 caractères maximum)
    • Brève description - Description de l'événement (250 caractères maximum)
    • Balises - Balises à associer à l'événement ; les balises seront attribuées à la vidéo si la vidéo est enregistrée en tant que ressource vidéo (128 caractères maximum et 1200 tags)
    • ID de référence - ID de référence pour l'événement ; doit être unique (150 caractères maximum)
  3. Tous les champs personnalisés requis pour le compte seront affichés. Affectez des valeurs aux champs personnalisés.
  4. La définition de la diffusion en continu sur les réseaux sociaux sur Autorisé rendra l'événement disponible pour diffusion en continu sur les plateformes de médias sociaux. Cela peut être fait une fois que l'événement a commencé à diffuser ou en utilisant le module Social.
  5. Sélectionnez un profil d'ingestion en direct. La liste déroulante répertorie tous les profils d'ingestion en direct. Le profil d'ingest contrôlera la qualité et le nombre de rendus en direct qui sont créés. Cliquez sur le lien Gérer les profils pour ouvrir l'éditeur de profil ingest dans un autre onglet du navigateur. Notez que vous devez être un administrateur de compte pour accéder à l'éditeur de profil ingest. Cliquez sur le lien FORMATS ASSOCIÉS pour afficher les formats associés qui seront créés à l'aide du profil d'ingest sélectionné.
    Les détails de débit et de facturation des trois niveaux sont les suivants :
    • Le niveau 1 permet jusqu'à 5 500 Kbits/s et est chargé à 1,0 heures
    • Le niveau 2 permet jusqu'à 10 000 Kbit/s et est rechargé à 1,5 heures.
    • Le niveau 3 permet jusqu'à 20 000 Kbps et est chargé à 2,0 heures
  6. Sélectionnez Convertir l'événement en actif vidéo une fois terminé pour enregistrer une version VOD de la vidéo en direct sur votre compte Video Cloud. Lorsque cette option est sélectionnée, le format associé de la plus haute qualité du profil d'ingestion en direct sélectionné est utilisé comme source pour la vidéo VOD. Le profil d'ingeste par défaut du compte est utilisé lors de la création de formats associés. Notez que la version VOD n'est pas disponible dès la fin de l'événement en direct, la vidéo sera disponible une fois le processus d'ingestion terminé.
    Note : Cette option n'est pas prise en charge lors de l'utilisation d'un événement récurrent.
  7. Cliquez sur OPTIONS AVANCÉES et configurez les options avancées.
    Note : Pour le travail Beacon Live, vous devez activer la case Notifications d'état de flux dans les options avancées.
  8. Activer les notifications d'état du flux

    Ajouter l'URL des notifications à partir de Beacon CMS (cette URL sera fournie par le support Brightcove)

  9. Cliquez sur Créer un événement pour démarrer l'événement en direct. La page Control Room s'ouvrira avec des détails sur l'événement en direct, y compris le point de terminaison de diffusion (URL RTMP) et le nom du flux qui seront utilisés par votre encodeur.

    Il faudra environ une minute pour traiter vos flux.

    Pour commencer à diffuser sur la plate-forme Live, vous aurez besoin d'un encodeur. Pour plus d'informations sur l'utilisation d'un encodeur, veuillez lire la documentation suivante :

    Étape par étape : Utilisation du module Live avec Telestream Wirecast

    Étape par étape : Utilisation du module Live avec Open Broadcaster Software (OBS)

  10. Dans le Salle de contrôle copier le  Identifiant de l'emploi » et le "Identifiant vidéo" du Evénement test panneau, nous aurons besoin des deux pour configurer l'événement Live dans les paramètres de la plate-forme Beacon.
  11. Ensuite, revenez à Beacon et cliquez dans l'onglet Événement, accédez à l'événement que nous y avons précédemment créé, et dans les paramètres Données de base, collez l'ID de travail de votre événement Live.

    Ensuite, allez au Ruisseaux onglet et créez un nouveau Stream, dans le Identifiant de la vidéo en streaming , collez le Identifiant de la vidéo de l'événement Video Cloud que nous avons créé auparavant.

    Remplissez les informations suivantes :

    • Sélectionnez le nombre d'appareils sur lesquels vous pouvez publier votre vidéo.
    • Modifiez le type de flux en : En direct
    • Et cliquez sur le bouton « Mettre à jour l'événement » avant de quitter la page :
  12. Effacez votre cache.

    Accédez à l'application Brightcove Beacon et sélectionnez votre playlist et votre événement. Cliquez sur Play.

    Vous devriez être maintenant en mesure de voir votre streaming en direct à partir de Video Cloud dans l'application Beacon.

    Une fois votre événement terminé, vous pouvez l'enregistrer en tant que clip pour le transmettre ultérieurement en VOD sur Beacon.

Créer un clip à partir de votre événement Beacon Live

Vous devez créer un clip de votre événement Live Beacon si vous souhaitez conserver la vidéo pour la diffusion ultérieure en tant que vidéo enregistrée en VOD.

Le module Live permet de créer des clips vidéo à partir d'événements terminés. Des clips peuvent également être créés pendant qu'un événement en direct est en cours.

Pour une liste complète des fonctionnalités d'écrêtage, veuillez consulter le Création d'un clip vidéo à l'aide du module Live Documentation.

  1. Dans le module Live, accédez à l'onglet Créer un élément dans le panneau de gauche
  2. À l'aide des boutons d'écrêtage ou des touches de raccourci, définissez les points de début et de fin de l'élément. Ou cliquez simplement sur le bouton Enregistrer pour conserver l'intégralité de la vidéo.
  3. Saisissez les propriétés de la vidéo du clip.
    • Nom de l'élément : nom de l'élément
    • Brève description - Description du clip
    • Tags - Balises à associer au clip
    • ID de référence - ID de référence à attribuer au clip
    • Supprimer le contenu des sauts publicitaires source : cette option supprime les sauts publicitaires déclenchés dans le clip, en conservant uniquement le contenu du programme. Remarque : La case à cocher Supprimer le saut d'annonce source s'affiche uniquement lorsque le flux est activé par SSAI.
  4. Cliquez sur Créer un élément. Le processus de transcodage de l'élément démarrera. Les clips sont transcodés à l'aide du profil d'ingeste par défaut du compte.

Ajoutez votre Clip à Beacon en tant que vidéo en flux lisible.

  1. Prenez l' ID numéro de votre clip créé :
  2. Dans la plate-forme Beacon, rendez-vous sur le Événements onglet et sélectionnez le Ruisseaux menu, modifiez le Type de flux dans le menu déroulant de Live to Stream, et dans le Identifiant de la vidéo en streaming option, collez le identifiant numéro que vous venez de copier à partir de votre événement Clip sur la plateforme Live.
  3. Cliquez dans l'onglet Données de base et changez le statut de Terminé à Terminé. Avant de partir, cliquez sur le bouton Mettre à jour l'événement en bas de la page, cela rendra le bouton Lecture disponible dans l'application pour voir l'enregistrement de l'événement.

(Point d'entrée statique) Evénements en direct SEP

SEP sont des événements récurrents que vous pouvez laisser activé/désactivé lorsque votre événement redémarre encore et encore comme pour un programme hebdomadaire, par exemple. Cet événement peut être utilisé comme un seul pour tous les Live Job ID pouvant être transmis à plusieurs canaux dans l'application Beacon.

  • Notez que lorsque vous créez un événement récurrent, une vidéo est automatiquement créée pour celui-ci dans le module Media. Ne supprimez pas cette vidéo, car elle sera nécessaire chaque fois que l'événement se reproduit.
  • L'événement récurrent doit être activé avant le démarrage de l'encodeur.

Pour créer un événement récurrent, sélectionnez l'option Créer cet événement en tant qu'événement récurrent à l'aide d'un point d'entrée statique (SEP) lors de la création d'un événement en direct, puis cliquez sur Créer un événement.

  1. Dans Live Platform, créez un nouvel événement. Pour plus d'informations sur la création d'un événement, veuillez lire la documentation suivante.
  2. Élargir la Options avancées , et activer le  Créez ceci comme un événement récurrent à l'aide d'un point d'entrée statique (SEP) » option. Pour plus d'informations dans SEP cliquez ici.
  3. Lorsque vous avez déjà un événement SEP, cliquez sur l' onglet Veille et vous pourrez l' activer/désactiver depuis la salle de contrôle en direct.
  4. Copiez votre ID de travail en direct
  5. Revenez sur la plateforme Brightcove Beacon et configurez vos événements en leur donnant un nom et une heure programmée.
  6. La date et l'heure de votre événement doivent être au format suivant :
  7. Cliquez dans le Événement de mise à jour bouton.

Vous pouvez effectuer ce processus avec tous les canaux d'événements que vous souhaitez sur la plateforme Brightcove Beacon, et tous seront diffusés à partir de la même transmission d'événement en direct, mais chacun avec la date et l'heure différentes que vous avez ajustées dans chacun de ses paramètres d'événement.
