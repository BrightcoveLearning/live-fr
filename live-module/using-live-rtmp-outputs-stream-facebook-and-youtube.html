Utilisation des sorties RTMP en direct pour diffuser sur Facebook et YouTube

Dans cette rubrique, vous apprendrez comment utiliser les sorties RTMP dans le module Live pour diffuser des événements en direct sur Facebook et YouTube.

Introduction

Le module Live peut être utilisé pour diffuser un événement en direct sur les ordinateurs de bureau et les appareils mobiles. Lorsque vous aurez saisi les informations sur l'événement dans le module Live, vous obtiendrez un ensemble de paramètres d'encodage à intégrer dans l'encodeur de votre site. Généralement, le flux en direct est publié à l'intérieur d'un lecteur Brightcove. La page Salle de contrôle du module Live permet d'entrer des URL de sortie RTMP supplémentaires. Les paramètres de sortie RTMP peuvent être utilisés pour envoyer le flux en direct aux sites de médias sociaux. Notez que les heures de sortie RTMP seront facturées en fonction des heures d'événement. Cette rubrique couvrira les étapes à suivre pour créer un événement en direct simple, puis montrer comment les sorties RTMP peuvent être utilisées pour diffuser un événement en direct sur Facebook et YouTube.

  • Les annonces, y compris les annonces côté serveur (SSAI), ne seront pas diffusées en continu vers les sorties RTMP. Le flux sans publicité sera utilisé à la place.
  • Les sorties RTMP sont limitées à 5 sorties par travail en direct. Cette limitation inclut les connexions marquées comme une erreur.

Création d'un nouvel événement

Pour créer un événement, suivez les étapes qui suivent.

  1. Ouvrez le module Live.
  2. Dans la page Gérer les événements, cliquez sur le bouton Créer un événement ou cliquez sur Créer un événement en direct ou sur Créer un canal en direct dans la navigation de gauche. La page Créer un événement en direct ou Créer une chaîne en direct s'ouvre.
  3. Entrez le nom de l'événement et les champs personnalisés requis.
  4. Sélectionnez un profil d'ingestion en direct.
  5. Sélectionnez Convertir l'événement en actif vidéo une fois terminé pour enregistrer une version VOD de la vidéo en direct sur votre compte Video Cloud.
  6. Cliquez sur OPTIONS AVANCÉES et configurez les options avancées le cas échéant.
  7. Cliquez sur Démarrer la diffusion en continu pour démarrer l'événement en direct. La page Control Room s'ouvrira avec des détails sur l'événement en direct, y compris le point de terminaison de diffusion (URL RTMP) et le nom du flux qui seront utilisés par votre encodeur.
    Note : Le traitement de vos flux prendra environ une minute.
    Note : Par défaut, l'encodeur doit être démarré dans les 30 minutes suivant la création d'un nouvel événement.
  8. Commencez à diffuser à partir de l'encodeur et le flux en direct apparaîtra dans la page de la salle de contrôle.

Une fois l'événement en direct démarré, le flux en direct peut être envoyé sur Facebook et/ou YouTube.

Note : Assurez-vous que l'encodeur est fermé/arrêté lorsque le flux en direct est terminé. Le fait de laisser une connexion ouverte entre n'importe quel encodeur et Brightcove, même lorsqu'aucune donnée n'est envoyée, peut entraîner des frais de connexion inattendus.

Streaming sur Facebook Live

Facebook Live est une fonctionnalité de streaming vidéo en direct sur Facebook qui vous permet de diffuser un événement en direct à votre public via la page de votre entreprise ou votre profil personnel. Les vidéos en direct apparaîtront dans les flux d'actualités d'un individu pendant l'événement en direct, ainsi qu'après la fin de l' événement.

Note : Pour plus de détails sur la diffusion en direct de Facebook, consultez la documentation Facebook.

Une fois le streaming en direct démarré dans le module Live, suivez ces étapes pour utiliser le flux en direct dans Facebook.

  1. Connectez-vous à Facebook.
  2. Créez un nouveau message et cliquez sur le Vidéo en direct option.
  3. Facebook peut vous demander d'accéder à votre appareil photo et micro. Cette rubrique utilise un appareil photo reflex numérique externe pour l'événement, alors cliquez sur Connect.
  4. Facebook affichera une URL du serveur et une clé de flux. Ces valeurs seront utilisées dans le module Live.
  5. Copiez l' URL du serveur dans le Presse-papiers.
  6. Dans le module Live, collez l' URL du serveur dans le champ URL de sortie RTMP de la page Salle de contrôle.
  7. Copiez la clé de flux de Facebook dans le Presse-papiers.
  8. Dans le module Live, collez le Clé de flux à la fin de URL de sortie RTMP champ. Le champ URL de sortie RTMP sera l'URL du serveur Facebook avec la clé de flux ajoutée, le tout sous la forme d'une seule valeur.
    Noter la seul format actuel pris en charge pour les URL RTMP est rtmps://host:port/application/streamName.
  9. Sélectionnez un rendu dans le Sélectionnez le rendu liste déroulante et cliquez Ajouter une sortie RTMP.
  10. Sous la section RTMP, confirmez que la connexion a réussi.
  11. Confirmez que le flux en direct apparaît dans Facebook. Entrez un titre et une description pour Facebook.
  12. Cliquez sur Aller en direct.

Le flux en direct commencera sur Facebook.

Cliquez sur Terminer pour mettre fin au flux en direct Facebook. N'oubliez pas d'arrêter l'encodeur lorsque l'événement est terminé.

Streaming sur YouTube

YouTube offre la possibilité de diffuser des vidéos en continu en direct à votre audience. Avant de commencer un streaming en direct pour la première fois, vous devez activer votre chaîne YouTube.

Note : L'activation d'un flux en direct pour la première fois peut prendre jusqu'à 24 heures. Une fois activé, votre flux peut être diffusé instantanément.
Note : Pour plus de détails sur la diffusion en direct YouTube, consultez la documentation YouTube.

Une fois le streaming en direct démarré dans le module Live, suivez ces étapes pour utiliser le flux en direct sur YouTube.

  1. Connectez-vous à votre compte YouTube.
  2. Accédez à la page YouTube Studio à l'adresse https://studio.youtube.com.
  3. Dans la navigation de gauche, cliquez sur Autres fonctionnalités en > direct.
  4. YouTube peut vous demander d'accéder à la caméra et au micro de votre ordinateur. Cette rubrique utilise un appareil photo reflex numérique externe pour l'événement, alors cliquez sur Connect.
  5. YouTube affichera une URL du serveur et un nom/clé de flux. Ces valeurs seront utilisées dans le module Live.
  6. Copiez l' URL du serveur dans le Presse-papiers.
  7. Dans le module Live, collez l' URL du serveur dans le champ URL de sortie RTMP de la page Salle de contrôle.
    Noter la seul format actuel pris en charge pour les URL RTMP est rtmps://host:port/application/streamName.
  8. Copiez le nom/clé Stream de YouTube dans le presse-papiers.
  9. Dans le module Live, collez le Nom/clé du flux à la fin de URL de sortie RTMP champ. Le champ URL de sortie RTMP sera l'URL du serveur YouTube avec le nom/clé de flux ajouté, le tout sous la forme d'une seule valeur.
  10. Sélectionnez un rendu dans le Sélectionnez le rendu liste déroulante et cliquez Ajouter une sortie RTMP.
  11. Sous la section RTMP, confirmez que la connexion a réussi.
  12. Confirmez que le flux en direct apparaît dans YouTube.

Le flux en direct commencera sur YouTube.

Cliquez sur Return to YouTube Studio, puis arrêtez l'encodeur pour mettre fin au flux en direct. Une fois votre flux terminé, une vidéo publique sera automatiquement créée et téléchargée pour que les utilisateurs puissent les visionner ultérieurement.
