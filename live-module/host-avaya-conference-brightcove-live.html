Introduction

Avaya Rooms Systems est une application de visioconférence qui vous permet de rencontrer d'autres personnes en ligne. Pour toucher un public plus large, vous pouvez tirer parti de Brightcove Live pour diffuser votre réunion pour que d'autres personnes puissent le voir.

Étapes

Procédez comme suit pour organiser une réunion sur Brightcove Live :

  1. Dans Video Cloud Studio, créez un événement Live à l'aide du module Live.
    Si vous souhaitez enregistrer la diffusion une fois qu'elle est terminée, assurez-vous de vérifier la Convertir l'événement en élément vidéo une fois terminé case à cocher !

  2. Une fois que vous avez créé l'événement Live, copiez l'URL RTMP et la clé de diffusion en continu depuis le panneau Salle de configuration.

    • URL RTMP en streaming - URL du point de terminaison en streaming (Vous pouvez trouver l'URL du point de terminaison dans la salle de contrôle Brightcove Live Control Room
    • Clé de streaming - Nom du flux (dans Brightcove, le nom du flux est toujours  vivant »)
  3. Publiez et intégrez le lecteur pour cet événement Live dans votre site Web/Intranet, et notez l'URL de la page.

Configurer la session de diffusion en continu

Avec l'URL RTMP Streaming et la Clé Streaming, vous devez les configurer sur l'unité de codec Avaya.

  1. Dans l'interface graphique Avaya, accédez à CONFIGURER > STREAMING.

    (Facultatif) Sur le site Web d'Avaya, accédez à PARAMÈTRES ADMINISTRATEUR > UTILITAIRES > STREAMING.

  2. Insérez votre URL RTMP en continu.
  3. Insérez votre clé de diffusion.

Démarrer/arrêter la session de diffusion

Pour démarrer ou arrêter la session de diffusion en continu, procédez comme suit :

  1. Allez à la PAGE D'ACCUEIL > PLUS DE > STREAMING.
  2. Clique le Diffusion bouton. Cela démarrera le flux en direct.
  3. La diffusion RTMP est activée par défaut. Pour le désactiver, dans la plate-forme Avaya, allez à CONFIGURER > ADVANCED > UTILITIES > STREAMING.
