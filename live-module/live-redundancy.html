Activation de la redondance dynamique

Notes :

  • Chaque compte a une limite SEP (Static Entry Point) de 3, à l'exception de us-west-2 qui aura une limite de 10. Si vous avez besoin d'une augmentation, veuillez contacter le support technique Brightcove.
  • Si vous utilisez la redondance avec SSAI, vous devez insérer des annonces dans tous les emplois du groupe en même temps.

La fonction de redondance en direct est activée en ajoutant deux flux ou plus à votre événement dans Brightcove Live Platform.

Commencez par créer un événement en direct. (Pour plus d'informations sur la façon de créer un événement en direct, veuillez cliquer ici).

  1. Allez dans les Options de flux, ces options vous permettront de nommer et de sélectionner la région de vos flux. Cliquez dans le (+) Ajouter un flux de sauvegarde bouton.

    Si le bouton Ajouter un flux de sauvegarde ne s'affiche pas, le streaming de sauvegarde n'a pas été activé pour votre compte.

    Note : Veuillez contacter le support Brightcove pour activer cette option dans votre compte.
  2. En cliquant sur le Ajouter un flux de sauvegarde bouton, vous ajouterez un flux de sauvegarde (redondance) pour votre vidéo en direct. Vous pouvez en additionner autant que votre limite de compte prend en charge.
    Il se peut que des groupes redondants soient pris en compte dans la limite de votre compte. Une fois qu'un événement avec des flux redondants est terminé ou annulé, il n'est plus visible dans le Live Module. Cependant, ces événements peuvent être répertoriés et supprimés via l'API Live. Voir l'API Live : Groupes redondants.
  3. Nommez vos flux principal et de sauvegarde, puis sélectionnez la région la plus proche de l'emplacement de votre encodeur. Par défaut, la région de flux principal est définie sur Oregon (Ouest américain), ce qui produira de mauvais résultats si vous diffusez en continu depuis un autre continent.

    L' ID de référence est un identificateur facultatif pour usage interne et doit être unique (maximum 150 caractères).

    Continuez à configurer votre événement Live comme vous le ferez normalement. Pour plus d'informations sur la façon de créer un événement en direct, veuillez cliquer ici.

  4. Vous pouvez également ajouter un nouveau flux à partir de la fenêtre de la salle de contrôle :
  5. Ajoutez les informations et la région de votre flux de sauvegarde.
  6. Dans la salle de contrôle, vous pourrez voir les détails du point de terminaison et du nom de flux de chacun de vos flux.
  7. Vous pouvez consulter toutes les informations de flux dans la salle de contrôle.

Note : Si vous avez créé un événement à flux unique et que vous ajoutez un flux de redondance dans la salle de contrôle, vous devrez republier le code intégré de votre lecteur pour bénéficier de la redondance.

Coupure

  1. L'écrêtage vidéo sera disponible pour chacun de vos flux, simplement en sélectionnant le flux souhaité dans le menu déroulant. Pour plus d'informations sur la création d'un clip, reportez-vous à la section Création d'un clip vidéo à l'aide du module Live.
  2. Deuxième sélection de flux.

Surveillance

Cliquez dans l'onglet Surveillance sous le lecteur d'aperçu pour afficher les informations de surveillance. La surveillance sera disponible pour chacun de vos flux, simplement en sélectionnant le flux souhaité dans le menu déroulant.

Les informations de surveillance sont disponibles pendant qu'un événement est en cours et jusqu'à 1 heure après la fin de l'événement.

Suppression d'un groupe redondant

Avec un groupe redondant créé :

  1. Cliquez sur Annuler la diffusion d'un emploi du groupe.
  2. Cliquez sur Arrêter ce flux d'un deuxième emploi du groupe.
  3. Allez au Gérer les événements page et sélectionnez le Complété languette.
  4. L'événement de redondance devrait se terminer dans Inactif Etat.
  5. Cliquez sur proche pour supprimer l'événement de redondance.

Limitations

  • Redondance en direct ne peux pas être utilisé avec les flux en direct protégés par DRM.
