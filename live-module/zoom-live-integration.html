Intégration d'une réunion Zoom dans Brightcove Live

L'intégration de Brightcove Live Zoom vous permet de diffuser facilement des réunions Zoom à l'aide de Brightcove Live.

Conditions préalables

Sur la place de marché Zoom

L'intégration du zoom avec Brightcove Live est gérée par le Brightcove Live Connect app sur la place de marché Zoom. Il s'agit d'une application au niveau du compte destinée à être utilisée par des comptes Zoom au niveau du propriétaire ou de l'administrateur sous licence. Veuillez vous assurer que votre compte Zoom dispose des privilèges appropriés pour prendre en charge l'intégration.

Dans Brightcove Video Cloud

Avant de démarrer une intégration, vous devez disposer d'un champ de métadonnées personnalisé pour savoir si un événement est diffusé à partir de Zoom. Cela suivra l'état de l'intégration entre les sessions, c'est-à-dire si un événement est actuellement lié à une réunion Zoom. Pour plus d'informations sur l'ajout du champ de métadonnées personnalisé, veuillez consulter le documentation suivante.

  1. Aller à Video Cloud Studio > Administrateur > Champs vidéo > Ajouter un champ personnalisé
    2. .
  2. Écrire un Afficher un nom pour votre champ de métadonnées personnalisé.
  3. Pour le Nom interne champ, vous devez écrire : bc_meeting_id
    N'ajoutez aucun identifiant de zoom externe ni aucune donnée dans le champ personnalisé.
  4. Écrire un La description pour votre champ de métadonnées personnalisé.

Meilleures pratiques

Pour de meilleurs résultats, veuillez suivre les instructions ci-dessous.

  1. Utilisez le Démarrer et Arrêter Boutons de diffusion dans la salle de contrôle plutôt que dans le lecteur Zoom hôte.
  2. Évitez de diffuser plusieurs réunions Zoom en même temps sous un seul compte.

Créer une réunion Zoom

  1. Tout d'abord, accédez à votre Compte Zoom et créez une réunion Zoom. Pour plus d'informations sur la création d'une réunion Zoom, veuillez suivre le Documentation Zoom.
  2. Dans Zoom, accédez à Paramètres > Réunion (avancé) > réunions de diffusion en direct.
  3. Assurez-vous que les paramètres de votre compte Zoom sont Service de diffusion en direct personnalisé activée.

Intégration du compte Zoom avec Brightcove Live

Pour intégrer un compte Zoom à Brightcove Live, procédez comme suit :

  1. Connectez-vous à Video Cloud Studio.
  2. Clique le Administrateur bouton dans l'en-tête de navigation.
  3. Sélectionnez le Intégrations option dans le menu déroulant.
  4. Clique le Ajouter une intégration pour lier un ou plusieurs comptes Zoom à Brightcove Video Cloud.
  5. Une fenêtre contextuelle demandant l'accès à votre compte Zoom apparaîtra. Clique le Autoriser bouton.
  6. Fournissez vos informations d'identification Zoom pour intégrer le compte.

Ajout de l'intégration du zoom à un événement Brightcove Live

Pour ajouter l'intégration Zoom à l'événement Brightcove Live, procédez comme suit :

  1. Accédez au Module en direct dans Video Cloud.
  2. Clique le Créer un évènement bouton sur le Live Gérer les événements page. Pour obtenir des informations complètes sur la création d'un événement en direct, veuillez consulter le Création et gestion d'événements en direct à l'aide du module en direct Documentation.
  3. Remplissez les détails de l'événement en direct comme d'habitude, jusqu'à ce que vous atteigniez le Intégrations section.
  4. Clique le Ajouter une intégration bouton.
  5. Vous pourrez voir une liste de vos réunions Zoom actives et programmées.
  6. Cliquez sur le menu déroulant.
  7. Sélectionnez votre réunion. Dans cet exemple, je choisis le Réunion de documentation de test.
    Une seule intégration peut être ajoutée à un événement à la fois. Pour effacer l'intégration, cliquez sur le Supprimer l'intégration bouton. Si vous souhaitez avoir plusieurs comptes, vous pourrez le faire dans le panneau Admin > Intégrations.
  8. Une fois prêt à créer l'événement en direct, cliquez sur le Créer un évènement bouton.
  9. Dans la Control Room, il y a une option à onglets permettant de diffuser votre réunion depuis Zoom. Si vous préférez ne pas utiliser la source d'intégration, revenez à l'onglet Ingest URL dans la partie supérieure gauche.
  10. Si votre réunion Zoom a été créée récemment, essayez à nouveau de la synchroniser avec Zoom à l'aide du Synchroniser bouton.
    Si votre réunion ne s'affiche toujours pas après la synchronisation, essayez de vérifier que votre réunion est Programmé dans Zoom et que vous avez moins de 30 rendez-vous programmés.
  11. Dirigez-vous vers votre Réunion Zoom Compte.
  12. Cliquez sur le bleu Démarrer bouton pour démarrer votre réunion.
    En raison d'une limitation de Zoom, avant de commencer à diffuser la réunion Zoom sur Brightcove Live, la réunion Zoom doit être démarrée.
  13. De retour dans le Brightcove Live Salle de contrôle , clique le Commencer à diffuser bouton.
  14. Après la dernière étape, vous pourrez voir votre réunion Zoom dans la salle de contrôle.
    • À tout moment pendant la réunion Zoom, vous pouvez arrêter et redémarrer la diffusion. Cependant, si vous ne diffusez pas activement, l'événement peut expirer si l'intervalle dépasse le temps de reconnexion de l'événement.
    • La diffusion du zoom sur Brightcove sera retardée d'environ 20 secondes.
    • Si la réunion Zoom ne démarre pas la diffusion après 20 secondes, vous pouvez cliquer sur Arrêter la diffusion et alors Commencer à diffuser de nouveau.
    • Étant donné que le flux Zoom est un service tiers, certaines fonctionnalités avancées de Live peuvent ne pas être prises en charge, y compris SSAI.

Désinstallation de l'intégration Zoom

Supprimer l'intégration de Video Cloud

Pour supprimer les intégrations Zoom de votre compte dans Brightcove :

  1. Connectez-vous à Video Cloud Studio.
  2. Clique le Administrateur bouton dans l'en-tête de navigation.
  3. Sélectionnez le Intégrations option dans le menu déroulant.
  4. Clique le Poubelle icône.
  5. Lorsqu'on vous demande de confirmer, cliquez sur le Confirmer bouton.

Pour supprimer Brightcove Live Connect de votre compte Zoom

Vous pouvez supprimer complètement le Connexion en direct de Brightcove app depuis votre compte Zoom en vous connectant à Zoom :

  1. Connectez-vous à votre compte Zoom et accédez au Zoom App Marketplace.
  2. Cliquez sur Gérer> Applications installées ou recherchez le Brightcove app.
  3. Clique le Brightcove app.
  4. Cliquez sur Désinstaller.
