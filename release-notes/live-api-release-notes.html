Cette rubrique décrit les mises à jour qui ont été apportées au module Live et à l'API Live.

2021-06-07

  • Il est désormais possible pour les utilisateurs de programmer un clip vidéo à l'aide du module Live.

    En savoir plus

22/03/2021

  • Il est désormais possible pour les utilisateurs de planifier et de gérer l'heure de début et de fin d'un ou de plusieurs événements Live Job.

  • Les couleurs et les polices utilisées dans Brightcove Live ont été mises à jour pour correspondre à la nouvelle marque Brightcove.
