Audio multilingue

Cette fonction permet d'acquérir plusieurs langues audio, des pistes d'expérience audio descriptives ou ambiantes pour être disponibles pour la lecture en direct.

Prérequis :

  • Définir le PID (identificateur de paquet) pour chaque piste audio de votre encodeur.
  • Pour le moment, seul le format d'entrée RTP est pris en charge. Pour que cette section soit activée dans le Créer un événement en direct page, allez à la OPTIONS AVANCÉES section et sélectionnez Protocole de transport en temps réel (RTP) sous le Format d'entrée sélecteur.
  • Dans le Liste blanche CIDR section, entrez l'IP : 0.0.0.0/0 pour vous connecter au point de terminaison RTP. Le bouton multilingue sera désactivé si le format d'entrée n'est pas défini sur RTP.
    • Notes :
    • Pour plus d'informations sur la définition du PID (Packet Identifier), pour les pistes audio, reportez-vous à la documentation de votre Encoder.
    • La prise en charge de SRT sera ajoutée dans un proche avenir.
    • Pour plus d'informations sur la configuration d'une entrée RTP, veuillez visiter le Configuration de la documentation des options avancées.

Ajout de pistes audio à un événement en direct

Lors de la création d'un nouvel événement Live :

  1. Ouvrez le Pistes audio section.
  2. Clique le Ajouter des pistes audio bouton.
  3. Remplissez les champs pour ajouter une nouvelle piste audio. Les champs marqués d'un astérisque (*) sont obligatoires.
    • Par défaut: Sélectionnez la piste audio par défaut.
    • Langue*: Tapez le code de langue de la nouvelle piste audio. Exemple : en, es, Cy-GB, etc.
    • PID*: Entrez le code PID précédemment défini dans votre encodeur pour synchroniser les pistes audio de votre flux entrant dans la langue et la variante appropriées. Note : Des PID uniques sont requis pour chaque piste audio.
    • Variante: Représente la variante de la piste audio. Ceux-ci peuvent être : principal, commentaire, alternatif, légende, sous-titre, supplémentaire, dub.
    • Étiquette: Tapez un nom pour la piste. Exemple : Anglais, espagnol, etc. Notez que si le champ d'étiquette est laissé vide, le code de langue sera utilisé comme étiquette.
  4. En cliquant sur le + et - boutons, vous pouvez ajouter ou supprimer des pistes audio.
    Note : Vous pouvez ajouter jusqu'à six pistes audio. S'il est nécessaire d'ajouter d'autres pistes, veuillez contacter le support technique.

Changement de piste audio

  1. Si les pistes multi-audio ont été correctement configurées, le lecteur aura une icône de casque qui peut être utilisée pour basculer entre les pistes.
