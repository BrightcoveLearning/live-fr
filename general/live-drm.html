DRM en direct

Dans cette rubrique, vous apprendrez comment activer la protection du contenu pour votre flux en direct.

Activation de DRM pour le contenu en direct

Note :

Par défaut, DRM est appliqué aux sorties comme suit :

  • Votre compte Live doit d'abord être activé par le support Brightcove afin d'utiliser la DRM.

  • Pour publier un événement compatible DRM, utilisez l'une des méthodes suivantes :

    • Un lecteur Brightcove V6.37.1 ou version ultérieure compatible DRM
    • Le SDK natif Brightcove pour iOS v6.5.0 ou version ultérieure

    • Le SDK natif Brightcove pour Android v6.11.0 ou version ultérieure

  • Si vous souhaitez remplacer les paramètres par défaut de DRM, vous pouvez le faire via l'API ou contactez le support Brightcove pour les modifier dans Live Platform.
    • FairPlay : appliqué aux sorties HLS
    • Playready : appliqué aux sorties DASH
    • Widevine : appliqué aux sorties HLS[1-1]

Notes

  • [1-1] Widevine peut être appliqué aux sorties DASH au lieu de HLS. Pour faire de Widevine avec DASH la valeur par défaut nécessite un remplacement au niveau du compte. Contactez le support technique si vous souhaitez que cette dérogation soit appliquée à votre compte. En utilisant le API en direct vous pouvez également préciser widevine:hls ou widevine:dash pour des travaux particuliers, quelle que soit la valeur par défaut de votre compte. Notez que selon la façon dont votre compte/tâche est configuré, la réponse de l'API (comme celle illustrée ci-dessous) inclura widevine:hls ou widevine:dash les sorties, mais pas les deux.

Pour activer DRM pour votre contenu Live, procédez comme suit :

  1. Dans OPTIONS AVANCÉES > Content Chiffrement. Sélectionnez Gestion des droits numériques (DRM) dans la liste déroulante Chiffrement.
  2. Un montant supplémentaire de Expiration du jeton L'option apparaîtra, vous permettant de sélectionner le temps en secondes avec un délai d'expiration minimum de 900 secondes et un maximum de 2592000 secondes (le dernier égal à 30 jours).
  3. Dans le panneau d'information sur l'événement, vous pourrez voir les URL DRM pour Fairplay et Widevine.
  4. Pour publier un événement compatible DRM, cliquez sur Publier et intégrer > Configuration du lecteur dans la navigation de gauche.
    • Dans Options Live Player, sélectionnez un lecteur compatible DRM. (Pour la lecture DRM en direct, vous devrez sélectionner ou créer un lecteur compatible DRM v6.37.1 ou supérieur).
    • Si vous n'avez pas de lecteur DRM disponible, accédez à la section > Sélection du module du lecteur ou Créer une > lecture de lecteur dans la navigation de gauche. Sélectionnez le « Activer les DRM » option. (Le lecteur pour la lecture DRM en direct doit être v6.37.1 ou supérieur).
  5. De retour dans le module Live, vous pourrez sélectionner votre lecteur DRM et Publier votre événement DRM.
