Introduction

Le planificateur en direct vous permet de planifier des actions pour Emplois de point d'entrée statique (SEP).. Vous venez de configurer un flux de travail de planificateur avec les heures, et les actions API nécessaires seront gérées automatiquement. Il existe deux types de flux de travail du planificateur que vous pouvez configurer, comme détaillé dans les sections suivantes.

Démarrer / arrêter le planificateur

Le planificateur vous permet de configurer les heures d'activation et de désactivation d'un travail SEP, et vous pouvez également mettre à jour les heures. Voir les détails dans Planification de l'activation / désactivation d'un flux SEP.

Planificateur de clips

Vous pouvez également planifier la génération d'un clip à partir de votre travail Live, puis mettre à jour le calendrier ultérieurement ou annuler le clip si quelque chose change. Voir les détails dans Planification de la génération de clips pour un flux en direct.
