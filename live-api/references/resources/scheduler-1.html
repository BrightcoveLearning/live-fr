Offres d'emploi StartStop

Avec le point de terminaison JobStartStop, vous pouvez planifier l'activation et la désactivation d'un travail en direct existant. À l'heure spécifiée, notre système effectuera les appels API nécessaires pour activer et désactiver un Job en votre nom.

Conditions

  • La clé d'API Live de votre compte.
  • Le remplacement du planificateur doit être activé sur votre compte en direct.
  • Un Live Job existant avec "statique": vrai.

Créer un workflow JobStartStop

Supposons que nous voulions planifier un événement de 2 heures qui débutera au 2021-02-02T 14:00:00 + 00:00.

Les horodatages doivent être au format UTC sous forme d'horodatage unix en secondes. Vous pouvez utiliser https://www.unixtimestamp.com/ pour la conversion.

Notez que les limites de temps d'activation décrites ici s'appliquent toujours aux activations planifiées, nous vous recommandons donc de programmer une heure au moins 60 secondes avant l'heure de début souhaitée.

Pour cet exemple, nous choisirons 5 minutes avant l'heure de début prévue.

début : 2021-02-02T 13:55:00 + 00:00 = 1612274100 arrêt : 2021-02-02T 16:00:00 + 00:00 = 1612281600

Pour créer le workflow, faites la demande suivante

POST https://api.bcovlive.io/v1/scheduler/jobstartstop En-têtes : Type de contenu : application/json X-API-KEY : corps de votre clé API : { "job_id": "your Job ID", "activation": { "time_utc": 1612274100 }, « désactivation » : { "time_utc": 1612281600 }, « description » : « Mon émission du mardi »}

Vous devriez recevoir une réponse similaire à

{ "account_id": "your Account ID", "activation": { "state": "pending", "time_utc": 1612274100 }, « désactivation » : { "state": "pending", "time_utc": 1612281600 }, « description » : « My Tuesday broadcast », « job_id » : « votre numéro de poste », « type » : « jobstartstop », « workflow_id » : « d44cf29f4f184757ac0995beb8a0097e »}

Mettre à jour un workflow JobStartStop

Vous pouvez mettre à jour les heures d'activation et de désactivation d'un workflow JobStartStop. Ceux-ci ne peuvent être mis à jour que si "état": "en attente".

En suivant l'exemple précédent, nous souhaitons arrêter l'événement 30 minutes plus tôt que prévu.

METTRE https://api.bcovlive.io/v1/scheduler/jobstartstop/workflow_id En-têtes : Type de contenu : application/json X-API-KEY : corps de votre clé API : { "deactivation": { "time_utc": 1612279800 }}

Vous recevrez le flux de travail mis à jour dans la réponse.

{ "account_id": "your Account ID", "activation": { "state": "pending", "time_utc": 1612274100 }, « désactivation » : { "state": "pending", "time_utc": 1612279800 }, « description » : « My Tuesday broadcast », « job_id » : « votre numéro de poste », « type » : « jobstartstop », « workflow_id » : « d44cf29f4f184757ac0995beb8a0097e »}

Annuler un workflow JobStartStop

Vous pouvez annuler l'activation et la désactivation avec un EFFACER demander. Vous pouvez le faire après l'activation pour simplement annuler la désactivation.

SUPPRIMER https://api.bcovlive.io/v1/scheduler/jobstartstop/workflow_id En-têtes : X-API-KEY : votre clé API

Vous recevrez le flux de travail mis à jour dans la réponse.

{ "account_id": "your Account ID", "activation": { "state": "cancelled", "time_utc": 1612274100 }, « désactivation » : { "state": "cancelled", "time_utc": 1612279800 }, « description » : « My Tuesday broadcast », « job_id » : « votre numéro de poste », « type » : « jobstartstop », « workflow_id » : « d44cf29f4f184757ac0995beb8a0097e »}

Notifications

Vous pouvez éventuellement configurer des notifications lors de la création du flux de travail. Vous devez fournir une URL pour que notre service PUBLIER à. Nous enverrons une notification lorsque le planificateur effectuera les appels d'activation et de désactivation. Vous pouvez également configurer pour recevoir une notification m secondes avant l'activation et/ou la désactivation. Voir le Guide des notifications pour plus de détails.

POST https://api.bcovlive.io/v1/scheduler/jobstartstop En-têtes : Type de contenu : application/json X-API-KEY : corps de votre clé API : { "job_id": "your Job ID", "activation": { "time_utc": 1612274100, "notification": 600 }, « désactivation » : { "time_utc": 1612281600, "notification": 600 }, « description » : « Mon émission du mardi », « notification_url » : "http://example.com/live/jobstartstop/callbacks"}
